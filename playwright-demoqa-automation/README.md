
# Playwright DemoQA Automation

Proyecto de automatización para el reto técnico usando Playwright y TypeScript.

## Instalación

npm install

Instalar navegadores:

npx playwright install

## Ejecutar pruebas

npx playwright test

## Ejecutar con navegador visible

npx playwright test --headed

## Reporte

npx playwright show-report

## Casos cubiertos

- Formulario de registro
- Select menu
- Web tables (CRUD)
- Alerts
- Drag and Drop
